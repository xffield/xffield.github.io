TypeScript — JavaScript的超集

尝了 typescript 的甜头之后，JavaScript 已经回不去了。没办法，真的太香了。

TypeScript 是 Microsoft 开发的开源项目，是 JavaScript 的超集，提供了强大类型系统和代码检查。相比 JavaScript 使用 TypeScript 开发项目，大大增强了编码便利性、代码可读性和可维护性。类型定义完善，代码怎么改都不慌。

  • 类型系统是最好的文档。如函数的参数与返回值，类型声明下一目了然。
  • 编译阶段的代码检查，辅助检查代码规范。tsconfig.json、 TsLint、EsLint，按照团队约定进行配置。
  • 自带完备的 Language Service。VSCode 代码补全、接口提示、定义跳转，直接带飞。

需要声明一下的是 TypeScript 只是静态类型检查，检查出错仍然可以编译通过运行，编译成 JavaScript 之后就没 TypeScript 啥事了。

本系列最佳实践分为三个部分：

  • 类型声明，争取覆盖所有应用场景
  • tsconfig.json 常用配置
  • VSCode 相关插件推荐
