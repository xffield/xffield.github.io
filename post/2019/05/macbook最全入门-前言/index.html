macbook 是一款漂亮的电脑，搭载的 macOS 秉承着专注、流畅、效率的设计理念，让电脑的使用更加舒适。同时 macOS 也不像 windows 一样容易入门，需要经过一定的学习才能更好更流畅的使用它。如果你刚入手一台 macbook，操作macOS 可能还不是那么的顺手，那本系列教程应该会很适合你。

本系列教程目标是帮助不熟悉 macbook 的人熟练操作它，能够让我们最快、最高效的达成我们的目标。它有以下几个特点：

  1. 从 0 开始，系统性讲解，认知更全面
  2. 结合实际使用场景来操作，记忆更深刻
  3. 不断更新完善，保证时效性
